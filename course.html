Android Studio Features

Required Software:
You will need a Java Development Kit (JDK) before installing Android Studio.
Advantages:
Android Studio is a great all-on-one development tool for Android developers. It contains basically everything an Android developer would need, all in one convienent package.
Disadvantages:
The only disadvantage is possibly that it is only for Android. Also, it could be seen as having a complicated user interface for new users.
Available Languages:
All the major languages are supported.
Uploading & Selling:
Android apps are uploaded to the Google Play Store. Before uploading you will need to pay a one-time sign up fee of $25 to begin uploading.
3rd Party Emulator:
It is not needed to use a 3rd party emulator because Android Studio has one built in. But, if you want one there are some options out there, but it's definitely easier to stick to the built-in version.
